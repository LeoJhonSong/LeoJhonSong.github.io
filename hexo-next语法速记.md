### bs callout标签

用法:
```
{% note [class_name] %}
{% endnote %}
```

类型有:
- default (灰色右箭头)
- primary (紫色加号)
- success (绿色勾)
- info (蓝色info图标)
- warning (橙色感叹号)
- danger (红色减号)

[更多tag](https://hexo.io/docs/tag-plugins.html)

### pdf标签

[文档](https://github.com/theme-next/theme-next-pdf)

文件放在文章对应资源文件夹, 用法:
```
{% pdf blog/file.pdf %}
```
