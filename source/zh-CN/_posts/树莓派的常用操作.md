# 更改主机名

## 临时修改主机名：

```
sudo hostname 新的主机名
```

## 永久修改主机名：

主机名存储在两个地方，两个地方都有修改

sudo nano /etc/hostname

将原本的名称改为新的主机名

sudo nano /etc/hosts

将原本的名称改为新的主机名
