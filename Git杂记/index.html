有关SSH

用ssh方式登录git的话只是生成秘钥和在远程端添加秘钥麻烦一次, 后面就可以免密码, 而用http登录的话每次push都需要输入密码, 略微麻烦.

获取SSH Key

首先查看有没有~/.ssh这个文件夹以及文件夹下有没有id_rsa和id_rsa.pub两个文件. 如果有, 那就不用生成了.

这两个就是SSH Key的秘钥对，id_rsa是私钥，不能泄露出去，id_rsa.pub是公钥可以放心地告诉任何人. 比如加入GitHub的SSH Key.
